**Decode.io** is a cutting-edge web application designed to provide a seamless and modern user experience. Built with the latest web technologies, it features robust authentication, dynamic interfaces, and high-performance server-side rendering.

*   **Modern Tech Stack:** Built on **Next.js 15 (App Router)** and **React 19** for unparalleled performance.
*   **Beautiful UI:** Styled with **TailwindCSS** for a responsive and mobile-first design.
*   **Fluid Animations:** Integrated **Framer Motion** for smooth, professional transitions.
*   **Secure Backend:** Powered by **Supabase** for reliable database management and authentication.
*   **Icons:** Using **Lucide React** for crisp, consistent iconography.

## üõ†Ô∏è Installation & Setup

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/ayushsail/Decode.io.git
    cd Decode.io
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Set up Environment Variables:**
    Create a `.env.local` file in the root directory and add your Supabase credentials:
    ```env
    NEXT_PUBLIC_SUPABASE_URL=your_supabase_url
    NEXT_PUBLIC_SUPABASE_ANON_KEY=your_supabase_anon_key
    ```

4.  **Run the Development Server:**
    ```bash
    npm run dev
    ```

5.  **Open in Browser:**
    Visit [http://localhost:3000](http://localhost:3000) to see the app in action.
